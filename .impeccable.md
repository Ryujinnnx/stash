## Design Context

### Users
Stash serves AI dataset and model creators, researchers, and buyers who need to publish, discover, purchase, and manage access to high-value artifacts without centralized storage or payment rails.

### Brand Personality
Premium, technical, geometric. The interface should feel like serious marketplace infrastructure for high-stakes AI assets: precise, calm, and difficult to mistake for a generic SaaS template.

### Aesthetic Direction
Dark product UI using #080810 as the base and Electric Indigo #6366f1 as the primary action color. Typography uses Inter Tight for dense UI and JetBrains Mono for addresses, prices, metadata, and protocol-facing values.

### Design Principles
Keep data dense but scannable, expose on-chain state clearly, use motion only for state changes and flow continuity, avoid decorative noise, and make wallet/storage/access boundaries explicit.
